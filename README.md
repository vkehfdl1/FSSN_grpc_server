# Install
- install latest go-lang
- install latest protobuf for go-lang
- install latest gRPC for go-lang

# How to use

```shell
$ go build .\main.go
$ go run .\main.go -server={server_code}
```

### server code
1. Unary Call
2. Bidirectional Streaming
3. Client-side Streaming
4. Server-side Streaming
